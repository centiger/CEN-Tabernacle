# CEN Temple Explorer v1.0

CEN Tabernacle v2.1.0 Explorer Engine 기반 성막·성전 독립 PWA입니다.

## 포함 메뉴
- 성막
- 솔로몬 성전
- 스룹바벨 성전
- 헤롯 성전

## 현재 반영
- 화면 터치: UI 표시/숨김
- 핀치 확대/축소
- 드래그 이동
- 하단 탐험 메뉴
- 성전별 탐험 단계 데이터

## 주의
솔로몬/스룹바벨/헤롯 성전 이미지는 현재 임시 공통 성전 이미지(stage-temple.jpg)를 사용합니다. 최종 초고해상도 이미지가 준비되면 각 scene의 img 경로만 교체하면 됩니다.
