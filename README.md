# CEN Tabernacle v1.1.0

새 UX 방향 반영판입니다.

## 핵심 변화
- 평소에는 이미지가 주인공이 되도록 UI 자동 숨김
- 화면 터치 시 필요한 메뉴만 잠깐 표시
- 하단 설명은 기본적으로 손잡이+요약만 보이는 Bottom Sheet 구조
- 확대 도구는 🔍 버튼 아래에 접어 둠
- 장소 이동은 하단 동선바와 좌우 화살표 중심
- 뒤로가기: 근접 → 한걸음 → 전체 → 홈
- 언약궤 그룹 묘사 검토 메모 반영

## 주의
언약궤 이미지는 아직 임시 장면입니다. 다음 이미지 제작 시 그룹은 서로 마주보고, 날개를 위로 펴 속죄소를 덮고, 얼굴은 속죄소를 향하는 기준으로 교체해야 합니다.
